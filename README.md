# Dora Health Connect

A mobile-friendly web app prototype to connect patients to nearby health personnel.

## Features
- Capture patient complaints and medical details.
- Collect home address and optional GPS coordinates.
- Request doorstep care, telemedicine, or emergency ambulance dispatch.
- Show available nearby health personnel.
- Maintain a local timeline of recent requests.

## Run
Open `index.html` directly, or serve with:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
